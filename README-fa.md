# ابزار هوشمند مشاوره مهاجرت (مبتنی بر هوش مصنوعی)

یک اپلیکیشن وب مدرن، Pixel-perfect و بدون نیاز به دیتابیس که برای ارزیابی شرایط مهاجرتی کاربران با استفاده از مدل‌های پیشرفته هوش مصنوعی طراحی شده است. این ابزار با تمرکز بر سرعت، حریم خصوصی و تجربه کاربری استثنایی ساخته شده است.

## 🚀 ویژگی‌های کلیدی

- **ارزیابی مبتنی بر هوش مصنوعی**: استفاده از مدل‌های زبانی پیشرفته برای ارائه مشاوره‌های حرفه‌ای و پیشنهاد "طرح جایگزین" (Plan B).
- **معماری Zero-Build**: فرانت‌اند با HTML/CSS/JS خالص و استفاده از CDNها (بدون نیاز به NPM یا Vite برای اجرا).
- **طراحی Glassmorphism**: رابط کاربری سطح بالا با جلوه‌های شیشه‌ای، سایه‌های نرم و انیمیشن‌های روان.
- **پشتیبانی چندزبانه**: پشتیبانی کامل از زبان‌های انگلیسی، فارسی و عربی با تغییر خودکار جهت صفحه (RTL/LTR).
- **تاریخچه بدون دیتابیس**: ذخیره تاریخچه ارزیابی‌ها مستقیماً در فایل‌های محلی جهت امنیت و جابجایی آسان.
- **تجربه تک‌صفحه‌ای (SPA)**: تمام فرآیند از پرسشنامه تا نمایش نتایج در یک صفحه سیال انجام می‌شود.
- **قابلیت‌های خروجی**: امکان کپی نتایج در حافظه یا دانلود آن‌ها به صورت یک گزارش متنی ساختاریافته.

## 🛠 پیش‌نیازها

- **PHP**: ^8.3
- **Composer**: آخرین نسخه
- **AI API Key**: دسترسی به یک API سازگار با ساختار OpenAI (تنظیم شده در فایل `.env`)

## 📥 مراحل نصب و اجرا

برای راه‌اندازی پروژه در سیستم محلی خود، مراحل زیر را دنبال کنید:

۱. **کلون کردن مخزن**:
   ```bash
   git clone <repository-url>
   cd Immigration-consultant
   ```

۲. **نصب وابستگی‌های PHP**:
   ```bash
   composer install
   ```

۳. **تنظیمات محیطی**:
   فایل نمونه تنظیمات را کپی کرده و اطلاعات هوش مصنوعی خود را وارد کنید:
   ```bash
   cp .env.example .env
   ```
   فایل `.env` را ویرایش کرده و موارد زیر را پر کنید:
   - `AI_BASE_URL`
   - `AI_TOKEN`
   - `AI_MODEL`

۴. **تولید کلید اپلیکیشن**:
   ```bash
   php artisan key:generate
   ```

۵. **اجرای سرور**:
   ```bash
   php artisan serve
   ```

سپس آدرس `http://localhost:8000` را در مرورگر خود باز کنید.

## 📁 ساختار پروژه

- `app/Http/Controllers/EligibilityController.php`: مدیریت منطق هوش مصنوعی و تاریخچه فایل‌ها.
- `resources/views/welcome.blade.php`: رابط کاربری اصلی تک‌صفحه‌ای.
- `lang/`: شامل فایل‌های ترجمه JSON برای زبان‌های EN، FA و AR.
- `storage/app/eligibility_history.json`: محل ذخیره محلی ارزیابی‌ها.
